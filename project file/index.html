Scenario Walkthrough

Follow these steps to validate ACLs & Flow Designer.

1

Initialize as Alice

Switch user in the top header to Alice (Project Manager).

Navigate to Project Table, click New, and create a Project. (Alice has full access).

2

Create & Assign Task

Still as Alice, go to Task Table 2, click New.

Create a task, set Assigned To = Bob, Status = New, and click Save.

3

Impersonate Bob & Test ACLs

Switch user to Bob (Team Member).

Verify that Project Table navigator is hidden. Go to Task Table 2 and open Bob's task.

Observe that Task Name and Description are locked (Read-only ACLs in action!).

4

Trigger Flow Automation

As Bob, set Status = In Progress and write Feedback in the Comments field, then click Save.

This triggers the Flow Designer flow! Task status will auto-complete and send approval to Alice.

5

Alice Approves Task

Impersonate Alice again. Navigate to My Approvals (badge shows pending approval).

Click Approve. The task will be fully resolved! Check the Task list and Flow logs.
